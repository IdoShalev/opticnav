Directory hierarchy:

/ard
	Java SE, Android
	Eclipse workspace

	Contains sources for Android ARDS (MOD Live, Nexus 5)

/ard-management
	Jave SE, Desktop
	Eclipse workspace

	Contains sources for ARD management software
	Communicates over network with ARDs and Glassfish, and maintains
	state required for networked ARD interaction.

/common
	Contains sources shared among multiple projects

/web
	Java EE, Glassfish

	Contains sources for deployable Glassfish applications
